# Contributing to Digital Ninja

Thank you for your interest in contributing to Digital Ninja! This document provides guidelines for contributing to the project.

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Digital-Ninja-v1.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`
5. Make your changes
6. Test your changes: `npm run dev`
7. Build to ensure no errors: `npm run build`
8. Commit your changes: `git commit -m "Add your feature"`
9. Push to your fork: `git push origin feature/your-feature-name`
10. Open a Pull Request
